# 🎵 Vibe Check — Mood to Music

Type how you're feeling and get matched with the perfect music vibe, complete with immersive visuals, curated quotes, and direct Spotify/YouTube links.

## Quick Start (Local)

**Prerequisites:** [Node.js](https://nodejs.org/) v18+ installed.

```bash
# 1. Clone or download this project, then cd into it
cd vibe-check

# 2. Install dependencies
npm install

# 3. Start dev server
npm run dev
```

Open `http://localhost:5173` — that's it!
